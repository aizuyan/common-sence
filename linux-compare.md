# linux 大小比较

### 比较大小
`-gt`,`-lt`是算数运算符，用于比较整数的大小
`>`,`<`是字符串比较符，用于比较字符串的大小

`[ ... ]`和`[[ ... ]]`都可以用于比较

```shell
if [ 9 > 100 ] ; then
	echo "greater than"
else
	echo "less than"
fi
```
执行结果，可以看出世界上比较的是`"9">"100"`

```
greater than
```

#### =
比较变量是否相等的时候一帮使用`[ = ]`，不要使用`[ == ]`，后者兼容性问题比较大，不同的shell可能出现未知的问题。
